# To-Do List Application (Python + Tkinter)

A simple and user-friendly desktop To-Do List application built with **Python** and **Tkinter**. This project allows users to add, view, and delete tasks.

---

## ✅ Features
- Add new tasks  
- Delete selected tasks  
- Display all tasks in a listbox  
- Toast-style popup messages for feedback  
- Responsive window resizing  
- Clean and simple GUI layout  

---

## ✅ How It Works
- The **to_do.py** manages task creation, deletion, and marking as complete.  
- The **GUI** (gui.py) provides the interface using Tkinter widgets.  
---

## ✅ How to Run the App
1. Install **Python 3.8+**  
2. Tkinter is included with most Python installations  
3. Run the program:
   ```bash
   python gui.py
